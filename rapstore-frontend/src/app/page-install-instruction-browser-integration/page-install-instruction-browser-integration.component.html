Install instructions for the RAPstore browser integration

In order to bring the apps provided by RAPstore on your device, some additional software is needed. The browser is not able to do all necessary steps (mainly because of security reasons). With the browser integration, which consists of an extension and a native application, you get the ability to download an app directly on to your IoT device, without any further knowledge about programming software for your device.

1. Prerequisites

  • OS: Unix-based (e.g. Ubuntu)

  • Browser: Firefox, Chrome or Chromium

  • Python

2. Download

  • Download zip archive Download

  • Store the archive in a program directory

  • Unzip the archive (you can delete the archive afterwards)

3. Installation

  • open a terminal

  • execute "cd your_install_path/src"

  • execute "python install.py browser", where browser is one of [firefox, chrome, chromium]

4. Finish

If you are installing for chrome or chromium you will need to restart your browser to enable the extension.
