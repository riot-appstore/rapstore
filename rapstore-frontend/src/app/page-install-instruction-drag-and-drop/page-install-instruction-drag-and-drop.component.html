Install instructions for drag and drop

In order to install an app on your IoT device, you can simply copy the binary file directly to the board. However, this drag and drop mechanism is only supported for a subset of boards. Have a look at the detail page of the board to know whether your board is supporting that mechanism.

1. Download your app's binary

Save the file in your default 'Download' directory

2. Open your file explorer

  • Navigate to your 'Download' directory
  • Open a new window or tab and navigate to the storage of the connected device

3. Drag and drop the binary file to the storage

4. Your app is ready to use

You can disconnect your device if the process has finished (e.g., if the device's LED is not blinking anymore)
